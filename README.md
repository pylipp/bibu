# `bibu`

A Bitbucket command line client written in Python3.

## Ideas

### bibu configuration

- use git config to store e.g. username

### Pipelines

- trigger
- status

### Pull requests

- add
- list
- review

## Dependencies

- `requests`
- `gitpython`
